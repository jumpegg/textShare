TextShare 사용법

TextShare 는 기본적으로

markdown 문법을 사용합니다.

Header 사용법

# 은 header 태그를 사용할때 씁니다.

입력

# H1
## H2
### H3
#### H4
##### H5
###### H6

결과

H1

H2

H3

H4

H5
H6
BlockQuote 사용법

> 은 BlockQuote 태그를 사용할때 씁니다.

입력

> test1

결과

test1
목록 사용법

* 은 순서가 없는 목록을 사용할때 씁니다.

입력

* test1
* test2
* test3

결과

  • test1
  • test2
  • test3

숫자. 은 순서가 있는 목록을 사용할때 씁니다.

입력

1. test1
2. test2
3. test3

결과

  1. test1
  2. test2
  3. test3
