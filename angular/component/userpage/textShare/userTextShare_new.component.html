TextShare 사용법

TextShare 는 기본적으로


markdown 문법을 사용합니다.


마크다운에는 여러가지 문법이 있습니다. 이곳에 모든 문법이 적혀있지 않으므로 더 자세히 알고싶으신 분은 markdown 검색을 권장합니다.


Header 사용법

# 은 header 태그를 사용할때 씁니다.

입력

# H1
## H2
### H3
#### H4
##### H5
###### H6

결과

H1

H2

H3

H4

H5
H6
BlockQuote 사용법

> 은 BlockQuote 태그를 사용할때 씁니다.

입력

> test1

결과

test1
목록 사용법

* 은 순서가 없는 목록을 사용할때 씁니다.

입력

* test1
* test2
* test3

결과

  • test1
  • test2
  • test3

숫자. 은 순서가 있는 목록을 사용할때 씁니다.

입력

1. test1
2. test2
3. test3

결과

  1. test1
  2. test2
  3. test3
code 입력

``` (백틱) 은 코드를 입력할 때 사용합니다.

입력

```console.log('hello world');```

결과

console.log('hello world');
가로선 입력

*** 은 가로선을 그을 때 사용됩니다.

입력

***

결과

TextShare
